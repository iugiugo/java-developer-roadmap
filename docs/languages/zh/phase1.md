# 第一阶段 - Java基础与核心技能（2-3个月）

## 编程逻辑与算法
- 问题分解
- **练习**：平均值、奇偶数、排序

## 环境配置
- 安装JDK
- IDE（IntelliJ或Eclipse）
- 基本debugging

## Java基本语法
- 程序结构（class, main）
- 命名规范
- 注释

## 数据类型与变量

## 输入与输出
- Scanner

- if, else if, else, switch

## 方法与作用域

- try/catch/finally
- Checked vs Unchecked
